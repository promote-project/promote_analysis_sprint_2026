# Get ready before the Sprint
This page lists a few steps to take to get ready for the big day.


1. Make sure you have registered to the event by adding your name, email and affiliation to the [participants list](https://docs.google.com/spreadsheets/d/1YyH23keVlTGwccVt-pEEn9unGV5RgsgIu_KJG66yaQU/edit?usp=sharing).

2. Join the PROMOTE Analysis Sprint Slack channel 
	* For members of NCAS, search for #promote_analysis_sprint
	* For externals, email Charlotte (charlotte.lang@reading.ac.uk) to receive an invite.

3. Access to JASMIN and group workspaces (gws)


	<div class="warn">
	  <span class="warn-icon">⚠️</span>
	  <div class="warn-text">
	    If you don't already have a JASMIN account and access to the CANARI and TerraFirma gws, 
	please keep in mind that it might take a few days to get your accounts approved.
	  </div>
	</div>

    * Sign up for a JASMIN account by following the instructions [here](https://help.jasmin.ac.uk/docs/getting-started/get-jasmin-portal-account/). Mention the PROMOTE Analysis Sprint in the **What will you be using JASMIN for?** box.
* Sign up for JASMIN services through the [accounts portal](https://accounts.jasmin.ac.uk/account/mfa/verify/?next=/):
		* JASMIN login account
		* PROMOTE, CANARI and TerraFirma gws. Apply via the 'Manage JASMIN services' button and mention that you need access to the gws for the PROMOTE Analysis Sprint.



4. We will be using Teams for presentations, highlights and communication during the sprint so please make sure you are able to connect before the event. You will receive Teams calendar invites for each day closer to the start date.
